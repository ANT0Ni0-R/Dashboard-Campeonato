# Fotos dos Closers (competições)

Fotos usadas no pódio dos dashboards de competição. Servidas via
`raw.githubusercontent.com` (este repo é **público**), consumidas pelo `Code.gs`.

## Convenção

```
competicoes/fotos/<PMP>.jpg
```

- `<PMP>` = código de 3 letras do Closer, **maiúsculas** (ex.: `EZB`, `HLM`).
- Extensão **`.jpg`** (o front monta a URL como `<base>/<PMP>.jpg`).
- Recomendado: ~600–900px no maior lado, recorte ~4:3, < 200 KB.

URL de exemplo (na `main`):
`https://raw.githubusercontent.com/ANT0Ni0-R/Dashboard-Campeonato/main/competicoes/fotos/EZB.jpg`
